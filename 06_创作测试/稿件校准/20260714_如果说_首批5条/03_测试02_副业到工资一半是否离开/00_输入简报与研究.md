## 2. 输入简报

- 原始选题：副业没有完全覆盖主业时，是否可以离开；用户认为副业达到主业收入一半时已经可以考虑离开
- 用户材料性质：**主观判断**，不是用户亲历辞职成功后的经验总结
- 目标观众：20—35 岁上班族、已经开始做副业或想建立第二条收入路径的人
- 内容目标：共鸣、信任和职业决策教育
- 核心论点：副业稳定达到工资一半时，不必机械等待百分之百覆盖才允许自己考虑离开；但百分之五十只是“进入离职审计”的信号，最终还要看收入质量、时间瓶颈、风险缓冲和失败可逆性
- 希望观众获得：把“敢不敢辞”改成一套可以填写、比较和复盘的离职判断
- 禁止材料：虚构用户已经辞职、创业成功或靠副业养活自己；把单月收入、一次爆单或单一客户写成稳定商业模式；把主观框架包装成普遍财务结论

## 3. 观点边界

1. `副业达到工资一半`是用户提出的观察锚点，不是研究验证的通用阈值。
2. `可以考虑离开`不等于`应该立刻裸辞`；本稿讨论的是何时进入严肃评估和过渡设计。
3. 工资覆盖率不能单独决定离职。副业净收入、必要生活成本、现金储备、家庭责任、社保医保衔接和复工能力都要单独计算。
4. 不提供个性化投资、税务或法律意见；具体社会保险、劳动合同、税务和经营合规按所在地与个人情况核对。

## 5. 研究母库与本条取舍

### 本条采用

- 混合创业的学习价值：先验证，再决定是否扩大承诺。
- 工作—副业角色冲突：留下主业也有机会成本，但冲突本身不能证明离职必胜。
- 最小商业闭环：需求出现、有人付费、完成交付、仍有复购或转介绍迹象。
- 离职审计四问：收入质量、时间瓶颈、风险缓冲、失败可逆性。

### 暂不采用

- 固定的“连续三个月”“六个月储备”等统一数字：缺少用户个人支出和责任信息，容易制造伪精确。
- 创业成功率、行业平均收入等宏观数字：与本条判断没有直接因果关系。
- 名人裸辞创业故事：容易把幸存者故事当成决策依据。

## 6. 对标校准调用卡

- 批次 / 测试：`20260714_如果说_首批5条 / 测试02`
- 主对标账号：如果说
- 主结构模型：`ifushuo.reframe-action.v1`（误解重构到行动）
- 开头家族：`ifs.opening.hidden-cost.v1`（具体账本与隐性代价）
- 开头三拍：副业到一半不必死等百分之百 → 剩余差距可能来自主业占用的核心时间 → 百分之五十究竟证明了什么、离开前还要审计什么
- 主情绪按钮：焦虑到掌控感
- 正文首次兑现：开头后立即区分“进入离职审计”与“明天裸辞”
- 代表正文示范：`全篇逐段示范（P01—P43）`
- 正文锚点：`7649729653048888612`
- 参考原稿：`7643072492197727503`（不上班的数字锚点）、`7644578524879129899`（主业作为底仓与副业进攻端）、`7647472332063264054`（现金流、试错成本与调头速度）
- 论证顺序：熟悉门槛翻面 → 承认兼职验证的价值 → 解释百分之五十的含义与局限 → 四问审计 → 不同答案对应不同选择 → 克制收束
- 段落呼吸：一至两句一段；每个抽象判断后接账本、订单、客户或家庭责任场景
- 情绪路线：被大胆观点截停 → 意识到不能只看工资覆盖率 → 看见自己当前处境 → 获得审计方法 → 恢复选择感
- 结尾功能：让观众完成一次离职审计，不鼓励冲动表态
- 用户真实材料：用户明确表示这是主观判断，允许以本人身份公开表达
- 事实核验状态：研究和政策来源已核验；百分之五十阈值、四问框架均明确为本稿观点工具
