# 测试02｜V0 / V1 / V2 / V3 / V4 代表段落对比与评分

## 1. 对比结论

V1 没有改变 V0 的核心立场：副业稳定达到工资一半时，可以进入严肃离职评估，不必机械等待百分之百；最终决定仍取决于闭环质量、时间增量和风险边界。

本轮主要改变开头兑现速度、论证顺序、场景位置、研究口播化、段落呼吸和结尾功能，属于严格同题校准。

## 5. 本轮可复用学习（暂定）

1. 职业与财务类强观点必须在开头同时给“主张”和“适用边界”，否则容易从锋利滑向鲁莽。
2. 研究不集中放在知识段；正方证据和反方证据分别出现在它们真正改变判断的节点。
3. 当用户只有主观判断、没有亲历故事时，可以公开说明观点性质，再用生活账本和可执行审计建立真人感。

以上三条仍是本次测试结论；用户随后明确提出的“禁止创作者自我免责句”已单独升级到个人语言画像，但没有写入通用 skill 或对标账号指纹。
