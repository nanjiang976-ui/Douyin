# 测试04｜V3.1与V3.2引用自然度对照

## 1. 判断标准

本轮不再用“有几句名言”判断完成度，而看四件事：

1. 名言出现以前，观众是否已经产生一个需要回答的问题。
2. 名言是否只压缩或推进一个已经存在的判断。
3. 名言出现以后，正文是否迅速回到生活动作、代价或选择。
4. 多句名言之间是否存在一条连续论证，而不是作者名单轮换。

## 2. 逐处对照

| 引用节点 | V3.1进入方式 | V3.2进入方式 | 自然度变化 |
|---|---|---|---|
| 《反对本本主义》两句 | 先宣布调查原则，直接报出两句原文，再补现实例子 | 先让观众看见“岗位、资格、工作日常都不知道，却已经认定上岸安全”的人；两句原文随后给这种顺序错误命名 | 理论由前置题词变成现实问题的答案 |

| “往者不可谏，来者犹可追” | 先报《论语·微子》出处，再说明过去不能替未来投票 | 先把去年投入放进复盘，再用原句完成过去到未来的转折 | 观点先成立，原句只负责换挡 |

## 3. 结构差异

### V3.1

`现代判断 -> 报出名言 -> 解释原意 -> 说明边界 -> 回到正文`

这个句型连续出现四次。单看每一处都说得通，连起来却像四张依次弹出的资料卡。

### V3.2

`现实错误显影 -> 核心原文命名 -> 真实验证 -> 研究处理反方 -> 四格问题出现 -> 名言压缩准备逻辑 -> 过去投入转向未来 -> 条件分流`

名言仍有五句，但它们已经嵌在同一条“调查—验证—承受失败—重新决定”的链上。
