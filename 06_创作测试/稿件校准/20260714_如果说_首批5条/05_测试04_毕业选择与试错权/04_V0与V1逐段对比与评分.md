# 测试04｜V0与V1逐段对比与评分

> 后验修正：本文件中的V1 91分是用户反馈前的内部判断。用户指出正文第一句难懂且有AI味、场景过多、观点不够犀利，V1后验评分修正为66/100。后续以`07_V1与V2逐段对比与评分.md`为准；本文件保留用于追踪机械审稿为什么没有发现听觉推进问题。

## 1. 对照结论

V1没有更换用户确认的选题D、开头C、两项研究、四格失败账、三种选择路径或结尾动作。变化集中在：让成绩公布后的生活继续向前、把家庭支持翻译成具体对话与外部名单、把关键反方放在宿命感出现的位置、让四格工具通过自然转向推进。因此这次对照属于严格同题校准，不是通过换题或增加新结论制造提升。

## 2. 控制变量

- 核心问题：普通家庭孩子毕业后，在考研、考公和工作之间，先算一次失败能不能接住。
- 用户确认开头：C，V0与V1不改字。
- 决策工具：钱、时间、信息、退路四格失败账。
- 主场景：成绩公布后的下一周。
- 外部证据：2026年大学生职业支持研究；2020年北京高校国内读研路径研究。
- 关键反方：以分数为主的考试通道能够削弱一部分家庭背景差异，普通家庭不应被写进宿命论。
- 最终动作：在备忘录中写完四格，并补全“如果失败，下周一我做什么”。

## 3. 代表性差异

| 位置 | V0处理 | V1校准 | 作用 |
|---|---|---|---|
| 第一次兑现 | 解释“失败账单”的定义与作用 | 先让校招结束、父母询问和房租到期同时进入下一周，再留下“我还能退回哪里” | 观众先想到自己的生活，再接收概念 |
| 不同失败成本 | 两类毕业生并列说明现金流与空档差异 | 在并列场景后加入同一图书馆里的注意力差异 | 家庭条件对备考的影响变成可见动作，不写成人格差异 |
| 家庭支持机制 | 研究结论后解释钱、信息、建议和情绪 | 研究后立即落到看职位表、模拟面试和“先回来住”的家庭对话 | 降低报告感，同时解释托底为何影响选择状态 |
| 外部补偿 | 概括信息、建议和情绪可以从外部获得 | 写成一张支持名单：规则、核对对象、商量对象和短暂停靠处 | 把“家庭给不了”从宿命判断改为可执行补位 |
| 关键反方 | 说明普通家庭不该被劝退 | 先指出只看家庭和只看分数分别漏掉什么，再回到下注方式 | 反方不只负责免责，还推动中心判断 |
| 钱与时间 | 完整解释每格要算什么 | 增加培训费是否真的补差距、日期节点和官方规则核对 | 工具从抽象分类变成能够今晚填写的内容 |
| 信息 | 询问目标、要求和真实日常 | 增加一张目标资料页，并区分“真想走这条路”和“逃开毕业不确定” | 帮观众检查动机与信息质量 |
| 三种路径 | 根据托底和目标清晰度分流 | 在脱产条件前加入一段稳定作息的真实测试 | 防止观众只凭一时冲动暂停工作机会和现金流 |
| 结尾 | 总结、升华、四格行动和评论选择 | 压缩重复解释，保留一句升华、一个行动、一个评论选择和系列签名 | 不使用三问，也不让账号签名替代全文总结 |

## 5. 原创性与个人语言检查

- V0与V1正文均使用`audit_cognition_draft.py`，带`ifushuo.language.adult-reframe.v1`和用户钩子库运行：零硬性失败、零警告。
- 只直接调用用户库短句`你信不信`；没有轻改或复制任何完整长钩子。
- 对标素材只迁移段落功能并保留素材ID、段落区间、来源和弃用内容；未迁移原稿中的人物、国家叙事、身份污名、比喻、收益率或原句。
- 自我免责、主观判断边界、`照出`、机械“第一第二第三”、结尾三问、创作者数据和空目录开场均未命中。
- V1只保留一处必要的`不是A，而是B`：`成熟不是敢赌，而是失败以后，生活还接得上。`它出现在成本、反方和工具均已证明之后，没有在开头重复使用。
- 章节标题只用于文档审阅；口播通过“先把成绩那天往后推一周”“再看两个毕业生”“说到这里”“现在画四格”等自然导航让结构可听见。

## 6. 密度与风险自检

- V0有效口播字符：约2250；预计约6分15秒—6分25秒。
- V1有效口播字符：约2290；预计约6分25秒—6分35秒。
- 信息密度：高。
- 填充风险：低；每一段分别承担场景、机制、反方、核对或行动。
- 听觉疲劳风险：中低；后半段四格工具信息密集，需要用户朗读判断停顿是否足够。
- 证据强度：中高；两项研究已核验且保留限制，决策工具明确为用户原创框架。
- V1最值得继续检查是否删除的句子：`四格写完，答案会开始分开。`它只承担听觉转场，是否保留以实际朗读为准。
