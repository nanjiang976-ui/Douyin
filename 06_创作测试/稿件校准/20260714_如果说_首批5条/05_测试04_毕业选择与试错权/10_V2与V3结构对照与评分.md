# 测试04｜V2与V3结构对照与评分

## 1. 对照目的

V2已经修复V1的作者调度、场景过多和观点不够明确。V3不把V2写成失败版本，只测试三项结构变化：用一个思想母题统领全文，把四格改造成贯穿式调查，让三种路径直接从调查结果中产生。

## 2. 结构对照

| 维度 | V2观点先行压缩场景版 | V3调查式决策版 |
|---|---|---|
| 主模型 | `ifushuo.pressure-action.v1` | `ifushuo.classic-translation.v1` |
| 开头 | C，承诺四格和三种选择 | C保持不变，作为控制变量 |
| 正文首次兑现 | 直接判断“把一次考试当成唯一出口最容易吃亏” | 直接提出“先完成调查，让答案出现在调查之后” |
| 贯穿线 | 目标成立 → 家庭风险 → 四格工具 → 条件分流 | 调查原则 → 调查机会与支持 → 四格调查 → 调查结果分流 |
| 思想母题 | 无 | `TM-001`“没有调查，没有发言权”贯穿 |
| 研究位置 | 两项研究集中解释家庭条件 | 一项处理考试宿命论，一项解释支持机制，均服务“还要查什么” |
| 场景 | 一个家庭支持例子 | 一个目标岗位说不清的例子 |
| 四格作用 | 后半段的决策工具 | 全文的决策调查方法 |
| 三种路径 | 每条路径分别展开成立条件和风险 | 作为四格调查结果的简洁分流 |
| 结尾 | 四格填写 + 评论选择 + 签名 | 补最大空白 + 下周一动作 + 评论选择 + 签名 |

## 3. V3真正减少了什么

1. 删除V2前半段“唯一出口—目标成立—沉没成本”三个连续判断模块的独立排放，把它们归入调查不足和时间格。
2. 家庭支持研究不再单独扩展成多个价值判断，研究后立即转成“缺什么支持、去哪里补”的调查任务。
3. 四格和三种路径不再各讲一轮相似条件；分流只读取四格结果。
4. 保留一个目标不清例子，不增加家庭、租房、图书馆、银行卡等近义场景。

## 4. V3新增的风险

1. “没有调查，没有发言权”辨识度很高，如果原始语境和现代翻译交代不清，容易像借名句压人。
2. `classic-translation`容易把正文写成概念课，因此V3必须在引文后立刻进入目标岗位和官方规则。
3. 两项研究仍然连续出现，虽各自任务不同，实际听感要由用户朗读判断。
4. V3比V2更统一，但第一中心判断没有V2“唯一出口”那么直接，需要观察哪一种更有记忆点。

## 6. 当前选择建议

- V2更适合：观众只想快速拿到明确判断和四格工具，观点力度优先。
- V3更适合：希望全文围绕一个方法推进，减少课程模块拼接，并让选题具有一个可回扣的思想轴。
- 当前推荐先审V3，但不覆盖V2。若用户觉得思想母题降低口语自然度或让开头后的答案变慢，可以保留V3的调查结构，删除具名引文；届时由于skill要求完整稿必须调用已核验母题，需要重新选择更自然的嵌入位置，而不是直接无记录删除。

## 7. 数据边界

- 两版评分都是发布前文本判断。
- V2尚未获得定稿确认，V3也尚未经过用户完整审阅。
- 不能据此推断真实留存、完播、评论或关注转化。
