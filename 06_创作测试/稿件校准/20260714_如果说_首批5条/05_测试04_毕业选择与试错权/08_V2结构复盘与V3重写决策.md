## 4. V3结构决策

### 主模型切换

- 主对标账号继续锁定：如果说。
- V2模型：`ifushuo.pressure-action.v1`。
- V3模型：`ifushuo.classic-translation.v1`。
- 选择理由：本题可以把“调查”从一个抽象方法翻译成毕业选择前的具体动作；它能够统一目标核对、规则查询、失败成本和退出安排，减少模块拼接感。

### 核心思想母题

- 核心母题ID：`TM-001`。
- 原话：`没有调查，没有发言权。`
- 出处：毛泽东《反对本本主义》，1930年5月；共产党员网原文已核验。
- 原始问题：面对实际工作问题，如果不了解现实情况和历史情况，凭什么先下结论。
- 本条现代翻译：在没有查清目标、规则、成本和退出安排前，先别替自己决定考研、考公还是工作。
- 使用边界：不把原文的历史政治语境偷换成人生鸡汤；只迁移“结论产生于调查之后”的方法，不用权威代替现实证据。

### V3全文路径

`保留开头C → 直接指出答案应从调查后产生 → 解释“调查”原始问题与现代用法 → 用一个目标不清的例子显影错误用力 → 考试相对开放的反方 → 家庭支持改变失败成本的机制 → 把四格改写成决策调查表 → 由调查结果决定三种下注方式 → 回扣调查与一个行动`

归一化宏观路径：

`judgment → thought motif translation → evidence and counterpoint → decision audit → conditional choice → callback action`

### V3只改变什么

- 保留用户确认的开头C，作为控制变量。
- 保留两项已核验研究、钱/时间/信息/退路四格和三种路径。
- 将四格从“后半段工具”提前升级为全文的调查方法。
- 将三种路径压缩为调查结果的条件分流，不再重复解释同一条件。
- 全文只保留一个目标不清的生活例子；研究之后立即翻译成判断，不增加连续场景。
- 结尾保留一个填写动作、一个评论问题和第四次行动实验签名。
