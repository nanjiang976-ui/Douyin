- 组合原则：
  - A2突出“争议判断—站队评论—内容承诺”。
  - B2突出“观看状态镜像—降低划走阻力—内容承诺”。

- oMel式观看状态只能在第一句同时命中当前选题时使用；不能先花数秒只谈视频形式。

## 3. 重组后的三个开头

### A2｜强截停版：争议判断 + 评论站队 + 内容承诺

- 主骨架ID：`UHOOK-025`
- 组件ID：`mirror` + `promise`
- 组件顺序：自测判断 → 互动 → 现实后果照见 → 本期内容承诺
- 直接调用/改写：`你是否同意这样一种说法`；`大家怎么看，把想法打在公屏上`
- 完整开头：

> 你是否同意这样一种说法：一份工作只要是你唯一的收入来源，它越稳定，你反而越不敢失去它？大家怎么看，把想法打在公屏上。你担心的可能不只是一份工资，而是工资一停，生活里就再也没有第二个支点。所以这期视频，我们就讲清楚：稳定能解决什么，为什么只有稳定还不够，以及普通人能握在手里的退路到底由什么组成。

- 三拍：唯一收入来源的争议判断 → 评论站队与失去第二支点的后果 → 稳定和退路分别解决什么
- 预计时长：约22.5秒
- 正文首次兑现：部门调整场景中说明“稳定仍在，人为什么已经开始慌”
- 12分评分：12/12
- 风险：第一句需要用“唯一收入来源”限定，不能扩张成“工作越稳定越危险”的绝对判断。

### B2｜强共鸣版：oMel式观看状态镜像 + 明确内容收益

- 主骨架ID：`UHOOK-011`
- 组件ID：`self_test` + `bridge`
- 组件顺序：观看状态与现实焦虑同句照见 → 承认划走冲动 → 给出三层内容承诺 → 留下自测
- 调用方式：拆分重组“我知道你现在可能正躺着刷视频”和“给我几分钟”的功能，不照搬完整长钩子。
- 完整开头：

> 我知道你现在可能正躺着刷视频，但一想到这份工作没了，下一份收入还不知道在哪里，心里就会发紧。看到这种几分钟的知识内容，你可能已经想划走了。但给我几分钟，我会把普通人的安全感拆成三个能落到现实里的支点：工资、缓冲，以及离开平台后重新获得收入的能力。听到最后，你会知道自己现在最该补哪一个。

- 三拍：观看状态与失去收入的焦虑同句命中 → 承认长知识内容的观看阻力 → 预告三层支点并留下自测
- 预计时长：约22秒
- 正文首次兑现：立即用部门调整场景解释第一层“工资与可预测生活”能保护什么。
- 12分评分：11/12
- 风险：观看状态并非所有人都符合；“可能”必须保留，且不能羞辱观众刷视频或保证改变人生。

### C2｜认知锋利版（推荐）：具体问题 + 评论动作 + 场景追问 + 内容承诺

- 主骨架ID：`UHOOK-019`
- 组件ID：`mirror` + `promise`
- 组件顺序：二选一自测 → 评论互动 → 账单场景照见 → 本期内容承诺
- 直接调用：`问你个问题`；`大家怎么想，可以把想法打在公屏上`
- 完整开头：

> 问你个问题：按时发工资的工作，和离开公司后还能挣钱的能力，哪个更接近安全感？大家怎么想，可以把想法打在公屏上。再往下想一步：如果工资明天停了，你最先担心的是房租、房贷，还是根本不知道下一份收入从哪来？所以这期视频，我们就把安全感拆开：稳定到底解决什么，退路又解决什么，以及你现在最该补哪一层。

- 三拍：具体二选一 → 评论动作与真实账单追问 → 稳定、退路和个人缺口的明确内容承诺
- 预计时长：约22秒
- 正文首次兑现：马上进入同一个部门调整场景，区分眼前稳定与变化后的选择能力。
- 12分评分：12/12
- 推荐理由：它补齐了用户指出的两个明确缺口，又保留原C的问题锋利度；互动不是空提问，后面的账单场景和正文都会回答同一个问题。
