# “如果说”账号文案说明书（65条终审稿定稿）

> 数据范围：65条作品，使用逐条人工终审稿作为正文唯一分析源；互动数据为2026-07-12采集快照。原始ASR、OCR和融合稿仅作为校对证据，不用于最终风格结论。

## 1. 可调用结论

“如果说”的核心能力不是说教，也不是单个金句，而是把一个说不清的现实痛苦加工成一条确定的认知路径：

> 命名痛苦 → 推翻旧解释 → 用传统概念/现代机制解释 → 放进生活场景 → 提前处理误解 → 给低门槛动作 → 用身份与情绪修复收束。

未来创作应模仿这条路径的结构、节奏和情绪落点。可复用原账号句子、人物故事顺序、标志性比喻或未核实的论据。

## 2. 三阶段，不是一种平均腔调

| 阶段 | 集数 | 采集快照点赞中位数 | 主要题材 | 文案气质 | 未来调用规则 |
|---|---:|---:|---|---|---|
| 早期试探 | 1—13 | 734 | 财富、社交、行动、商业、能力 | 现实压力强、概念密、结果导向、冲突快 | 仅用于赚钱意识、执行力、一人公司等硬题；降低羞辱、绝对承诺和夸大数字 |
| 过渡扩张 | 14—30 | 1,247 | 人物、历史、现实选择、方法论 | 叙事更强，开始用人物镜像解释规律 | 仅用于历史人物、现实故事、方法论选题；案例先独立核验 |
| 成熟爆发 | 31—65 | 22,496 | 情绪、关系、内求、传统智慧、人生境界 | 克制、诗性、温和但有力量、收藏型 | 默认主风格，服务认知、个人成长、自我提升 |

成熟期的中位点赞明显高于早期和过渡期。原因不是单纯“国学题材更容易火”，而是它完成了三项升级：减少高压训斥，增加观众自我代入；减少单点暴论，增加古语—场景—机制的连续解释；减少结果许诺，增加清醒、自由、边界与安定等情绪收益。

## 3. 目标观众与情绪任务

### 目标观众

- 正在内耗、焦虑、空虚、关系受困或方向不清的普通成年人；
- 想提升认知、改变自己、获得行动感，但不想接受空泛鸡汤的人；
- 需要更大的解释框架，来理解工作、关系、赚钱、坚持和人生阶段的人；
- 对传统智慧、历史人物、心理学解释有兴趣，但更关心它如何落到日常生活的人。

### 文案承担的四个任务

1. **命名**：把“我不行”改写成一个可理解的结构问题。
2. **解释**：让观众相信痛苦不是偶然，并看见其中机制。
3. **升维**：给出更高的身份、规律或视角。
4. **收心**：用一个问题、动作或诗性结尾，让观众恢复掌控感。

## 4. 选题与标题模型

### 成熟期高匹配母题

- 负面状态的重新解释：心烦、空虚、无反馈、生气、内耗并非单纯意志力问题；
- 传统概念的现代翻译：不争、弱、和光同尘、内求、知止、破相；
- 层级/阶段模型：情绪—价值—规律，强—智—觉—空，内观—内守—内定；
- 反向定义：慢、弱、不显摆、放下、顺其自然、当下，都不是表面理解；
- 关系与人生困境：期待、投射、身份焦虑、情绪防御、控制感；
- 可收藏的精神工具：一句判断、三心法、四步骤、六句话、十条总结。

### 标题公式

1. `为什么你越{行为}，反而越{负面结果}？`
2. `真正厉害的人，都懂得{反常识能力}`
3. `什么才是真正的{抽象概念}？`
4. `{古语/人物/概念}，点透现代人的{痛点}`
5. `{主题}的三个心法/四个阶段/六句话`
6. `你以为{表面优点}是好事，它可能恰恰在{隐藏代价}`

标题只制造解释缺口，不讲完答案；必须让正文能给出清晰、可核验的解释和边界。

## 5. 钩子、前30秒与论证节奏

### 成熟期前30秒的固定动作

1. 用可截屏的断言、古语、悖论或具体困境开门。
2. 立即指出观众的默认理解为什么不完整。
3. 放入老板、伴侣、父母、消息、工作、独处等日常镜像。
4. 明确本条要解决的唯一问题：看清、停止内耗、找到结构或建立判断。
5. 进入概念解释，不做自我介绍和宽泛铺垫。

### 论证链

> 强观点 → 生活镜像 → 古语/概念/人物 → 大白话翻译 → 反例或极端误读 → 短句重锤 → 行动问题 → 价值升华

这不是“先理论、再例子”的课堂结构。它通过抽象与具体交替，让观众约每20—40秒获得一次新的理解、情绪确认或短句回报。

### 节奏指纹

- 开头快：短句、高冲突、强定义。
- 中段稳：长句解释与8—16字短句交替。
- 每一段至少有一个转折：`不是X，而是Y`、`你越X，越Y`、`真正的问题在于`。
- 结尾慢：不再引入新概念，改为身份确认、判断问题、三段排比或低门槛动作。

## 6. 四个稳定模型ID

### `ifushuo.mature.misconception-reframe-action.v1`

适用：内耗、焦虑、行动、学习、创作与长期主义。

> 大众误解 → 反向结论 → 机制/古语 → 场景翻译 → 最强误解 → 一个动作 → 价值收束

### `ifushuo.mature.identity-ladder.v1`

适用：人生境界、决策、情绪、关系与成长阶段。

> 第一层本能反应 → 第二层抽离/规律 → 第二层局限 → 第三层松弛状态 → 同一场景重写 → 排比收束

### `ifushuo.mature.classic-modern-translation.v1`

适用：传统智慧、古语、处世、关系和人生认知。

> 概念/古语 → 流行误读 → 字面解释 → 现代场景 → 机制补充 → 边界 → 当代动作 → 回扣原句

### `ifushuo.transition.story-to-principle.v1`

适用：历史人物、现实故事、商业与方法论。

> 人物困境 → 关键选择 → 结果反转 → 规律翻译 → 当代同构场景 → 观众行动

早期只作为附加能力调用：`ifushuo.early.pressure-to-action.v1`，用于现实成本、执行力和赚钱意识，不作为认知账号默认语气。

## 7. 语言、反驳与结尾

### 可迁移语言功能

- 定义：真正、所谓、本质上、说到底；
- 反转：恰恰、反而、根本不是、不是X而是Y；
- 镜像：你有没有发现、你仔细想、回想一下；
- 过渡：这就引出了、听到这里、再往下看；
- 升华：规律、格局、本心、自由、清醒、通透。

### 反驳位置

- 开头承认观点听起来反常；
- 中段替观众问“那是不是躺平/冷漠/不努力”；
- 结尾前用“不是X，而是Y”收回到可执行的边界。

### 结尾模型

- 给一个未来可反复自问的判断问题；
- 确认观众不是失败，而是需要换一个解释；
- 用三段排比从焦虑推向自由、安定或清醒；
- 给一项低门槛动作：停三秒、问一句、记录一次、保留一个节奏。

## 8. 严格迁移边界

### 可以调用

- 选题的反向解释；
- 钩子的认知缺口；
- 长短句交替、反转位置、论证顺序和情绪路线；
- 对质疑的提前处理；
- 可收藏的模型和低门槛行动。

### 禁止直接调用

- 连续原句、独特比喻、人物故事顺序、个人口头禅；
- 未经独立核验的古籍、人物、心理学、神经科学、财务和投资结论；
- “绝对、保证、一定、唯一”等结果承诺；
- 将观众或他人固定羞辱为弱者、低级或没有价值。

## 9. 面向当前账号的调用优先级

当前账号主定位是认知、个人成长、自我提升。默认调用顺序：

1. `mature_31_65` 的误解重构、传统概念现代化、情绪修复和排比收束；
2. 仅当选题确实需要人物故事时，调用 `transition_14_30` 的故事镜像能力；
3. 仅当主题是赚钱意识、执行力、一人公司时，调用 `early_1_13` 的现实成本和行动钩子；
4. AI不自动出现；只有用户明确要求，且AI服务于认知主命题时，才能作为一个辅助场景。
