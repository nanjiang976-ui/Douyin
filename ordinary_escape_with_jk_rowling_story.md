# 普通人一定要学会“越狱”：加入 J.K. Rowling 真人故事版

## 1. 选题判断

- 选题：普通人一定要学会“越狱”
- 核心论点：普通人的第一步不是逆袭，而是从“不敢开始”的旧程序里拿回一点选择权。
- 新增故事：J.K. Rowling 在低谷中完成《哈利·波特与魔法石》，经历多家出版社拒绝后，最终由 Bloomsbury 出版。
- 插入位置：放在“习得性无助”解释之后，“所以普通人的第一步不是逆袭”之前。
- 使用理由：这个故事能证明“不是等状态完美才开始，而是先做出一个具体作品”。
- 风险边界：不要写成“你也能成为罗琳”，而是写成“她的故事提醒我们，先完成一个作品，本身就是越狱的第一步”。

## 4. 可直接插入的故事段

这一段建议插入在“但问题是，如果你一直这样想，你就真的会被困在原地。”之后：

```text
我想起 J.K. Rowling 的故事。
今天我们都知道她是《哈利·波特》的作者，但她开始写这个故事的时候，人生一点都不光鲜。
她经历过母亲去世、婚姻破裂、单亲育儿和经济压力。更现实的是，《哈利·波特与魔法石》的手稿后来也不是一拿出去就被认可，而是被多家出版社拒绝过。

你看，这个故事真正打动我的地方，不是“她后来成名了”。
而是她在还没有掌声、还没有资源、还没有确定结果的时候，先把脑子里的那个想法写完了。
如果她当时也一直等：等生活稳定一点，等自己更有信心一点，等别人先认可我一点，那可能就没有后来的故事了。

所以很多时候，越狱不是突然冲出去。
越狱是你在最想放弃的时候，仍然把一个想法，变成一个具体作品。
哪怕它一开始只是几页纸、一个文档、一段很粗糙的草稿。
```

## 5. 加入故事后的口播稿

普通人一定要学会“越狱”。  
但我说的越狱，不是让你裸辞，不是让你跟所有人对着干，也不是一冲动就把现在的生活全部推翻。

我说的越狱，是从那个每天焦虑、每天不满意、每天想改变，但又每天什么都没开始的自己身上，逃出来。  
为了干成这件事，我花了差不多十年。

以前我一直以为，困住我的是学历，是单位，是家庭条件，是机会不够，是别人不理解。  
后来我才发现，真正困住我的，很多时候不是外面的东西，而是脑子里那套默认程序。  
遇到机会，第一反应是：我不行。  
想做点事情，第一反应是：再等等。  
看到别人开始，第一反应是：他有资源，我没有。  
然后一年又一年过去。  
你看起来没有输给任何人，但你其实一直输给了那个“不敢开始”的自己。

心理学里有个词，叫习得性无助。  
简单说就是：当一个人长期觉得自己怎么做都没用，他后来就算真的有机会，也可能不会再伸手了。  
这就是很多普通人的状态。  
不是不知道要改变，不是不想赚钱，不是不想成长，而是过去一次次挫败，让你慢慢相信：算了，反正我也改变不了什么。  
但问题是，如果你一直这样想，你就真的会被困在原地。

我想起 J.K. Rowling 的故事。  
今天我们都知道她是《哈利·波特》的作者，但她开始写这个故事的时候，人生一点都不光鲜。  
她经历过母亲去世、婚姻破裂、单亲育儿和经济压力。更现实的是，《哈利·波特与魔法石》的手稿后来也不是一拿出去就被认可，而是被多家出版社拒绝过。

你看，这个故事真正打动我的地方，不是“她后来成名了”。  
而是她在还没有掌声、还没有资源、还没有确定结果的时候，先把脑子里的那个想法写完了。  
如果她当时也一直等：等生活稳定一点，等自己更有信心一点，等别人先认可我一点，那可能就没有后来的故事了。

所以很多时候，越狱不是突然冲出去。  
越狱是你在最想放弃的时候，仍然把一个想法，变成一个具体作品。  
哪怕它一开始只是几页纸、一个文档、一段很粗糙的草稿。

所以普通人的第一步不是逆袭，不是成功，而是重新拿回一点选择权。  
仔细想一想，今天能不能花十分钟，写下一个你最想改变的问题？  
能不能把收藏夹里的一条内容，整理成自己的笔记？  
能不能把一个想法，变成一段文字、一个视频、一个小作品？  
不要等全部想清楚再做。  
很多方向，不是坐在原地想出来的，是你一边做，一边撞出来的。

我做这个账号，也不是想告诉你我有多厉害。  
但我不想制造焦虑，恰恰相反，我会把我这些年踩过的坑、走过的弯路、积累下来的工具和方法，一点点拆开讲给你听。  
我会用接下来的 100 次行动实验，记录一个普通人如何从混沌里一点点走出来。  
这里会讲认知，讲赚钱意识，讲知识体系，讲自我改变，也会讲我正在用的工具和方法。  
这个账号最后能不能做成，我不知道。  
但我确定一件事：如果我走过的路，能帮你少绕一点弯，那它就有价值。

记得初中时有篇作文题目叫《现在就去做》。很多年以后我才明白，这句话不是鸡汤。  
这句话的字面意思是让你立刻、马上去做一个实际的事情。  
深层意思是：你不用等一个完美的自己出现，才开始改变。你是在开始改变以后，才一点点变成那个新的自己。  
说得通俗一点，就是从现在开始，先种下一颗种子，先让自己进入一种“我现在就可以改变一点点”的状态。

所以，从今天开始，普通人一定要学会“越狱”。  
首先做到的，就是从原来的自己身上，逃出来。  
不要喊口号，就从一个动作开始。

## 6. 金句库

1. 你看起来没有输给任何人，但你一直输给了那个“不敢开始”的自己。
2. 越狱不是突然冲出去，而是在最想放弃的时候，仍然把一个想法变成具体作品。
3. 不要等别人先认可你，先让你的想法有一个可以被看见的形状。
4. 普通人的第一步不是逆袭，而是重新拿回一点选择权。
5. 你不用等一个完美的自己出现，才开始改变。
