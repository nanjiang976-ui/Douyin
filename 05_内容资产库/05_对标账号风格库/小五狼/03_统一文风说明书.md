# 小五狼｜统一文风说明书

## 使用范围

本文以当前导入的 20 条视频文案为正文来源，账号名使用“小五狼”。账号定位可迁移到认知、个人成长、自我提升、行动力、关系与决策题材，不与 AI 绑定。

## 一句话文风

把一个让人焦虑、困惑或自我否定的问题，做成“听完就能看懂自己”的互动式认知课：先用问题让观众对号入座，再用必要的框架、概念和生活例子完成解释，最后把观众推回一个可执行的自我改变动作。

## 最擅长服务的痛点

- 明知该改变却做不到：内耗、自控、拖延、习得性无助、目标失约。
- 不知道怎样理解自己：聪明与认知、情绪、欲望、痛苦、独处。
- 面对关系和社交时缺少底气：PUA、向上社交、识人、关系依赖。
- 面对人生选择时没有框架：机会、经济学、环境、长期投入与放弃。

## 选题母题

1. **你以为的问题不是问题本身。** 先命名一个大众痛点，再给出更底层的解释。
2. **问题往往不只在能力，而在行为系统。** 把人格评判改为可训练能力。
3. **知识必须翻译成现实选择。** 心理学、哲学、经济学、传统概念都要落到日常场景。
4. **真正的成长，是停止背叛自己。** 从高压自省转向小承诺与持续行动。

## 固定推进结构

1. **问题开场**：结果承诺、二选一问题、可辨认的矛盾，或“你有没有发现”的自测。
2. **互动确认**：只在需要时用一到两个问题，把观众从旁观者变成当事人。
3. **目录预告**：仅在确有分类、步骤或机制时，说明将展开的要点。
4. **先给核心结论**：不把答案藏到最后，先建立“我会听懂”的信任。
5. **编号展开**：每一点按“概念/规律 → 大白话 → 日常场景 → 再确认”的顺序讲透。
6. **中后段翻转**：把对外部、他人或命运的关注，翻回自己能调整的选择、环境和边界。
7. **轻量行动与现实收束**：给一个低门槛动作，再回到可观察的选择或判断。

## 开头固定表达的调用方式

开头要单独选择一个表达家族，并完成三拍：**截停 → 让观众站队/照见自己 → 承诺解释或行动收益**。

本账号可调用的六个家族是：不公平比较自测、二选一后反转、具体收益加自测、风险警报与边界、熟悉概念现代翻译、人格问题系统化。具体句式骨架、情绪按钮、承接动作和禁用边界见 `05_开头表达指纹.json`。

“今天我们来讲”“在讲之前有两个问题”“把答案写在公屏上”只是原账号的课堂表层，不作为可复用钩子。问题必须制造真实选择或矛盾；收益必须具体、克制，并在正文第一部分兑现。

## 语言与节奏

- **提问服务理解**：用“你有没有”“你想想”等把抽象知识送回观众处境；未来迁移时要降低课堂确认句密度。
- **口语化翻译**：概念出现后迅速跟“什么意思”“换句话说”“说白了”拆开讲。
- **先长后短**：先用一段完整解释铺逻辑，再用短断句确认结论或制造顿挫。
- **重复必须换功能**：同一观点只有在概念、例子、反问、结论承担不同任务时才保留；否则删掉。
- **编号感强**：多点清单让长视频始终有进度条，适合讲认知知识而不显得散。

## 情绪路线

`被戳中/对号入座 → 得到解释 → 觉得自己可以掌控 → 愿意做一个小动作 → 看见可执行的下一步`

这套路线的关键不是单纯给安慰，而是先让观众感到“原来问题可以被看懂”，再给“我还能做什么”的确定性。

## 结构模型

### 1. 问题－框架－清单模型

适用：内耗、自控、认知提升、行动力。

`强问题/结果承诺 → 必要的互动自测 → 3-5 点框架 → 每点讲机制与场景 → 小动作 → 现实判断`

### 2. 概念－实验－生活模型

适用：心理学、脑科学、经济学、习惯与决策。

`痛点反问 → 概念命名 → 研究或理论作为解释支点 → 日常例子 → 可观察行为 → 行动建议`

### 3. 悖论－重新命名－行动模型

适用：高智商困境、痛苦、拖延、内耗、关系。

`表面矛盾 → 反直觉解释 → 让观众承认代价 → 重命名问题 → 可执行的替代选择`

### 4. 熟悉概念－现代翻译－决策模型

适用：传统智慧、环境、人生原则、长期主义。

`熟悉但有争议的概念 → 现代语言翻译 → 多场景验证 → 去宿命化 → 现实决策动作`

## 可迁移能力

- 用“高回报承诺 + 自测问题”取代平铺直叙的知识开头。
- 以 3-5 个强相关的小标题承接长视频，而不是堆砌道理；短视频没有真实分类时不凑编号。
- 每讲一个抽象概念，立刻补一段普通人的工作、关系、独处或选择场景。
- 用“不是 X，而是 Y”的翻转把焦虑转成可观察、可调整的机制。
- 结尾优先给一个当天能做的小动作，不只给大道理。

## 可直接调用边界

- 可复用原账号的连续原句、固定口头禅、英文夹杂、结尾口号或“课堂”人设。
- 可复用原视频的个人经历、人物背书、实验故事、案例顺序和具争议的判断。
- 不把观众贴为高低等级，更不能用贬损语言强迫共鸣。
- 不把心理学、脑科学、传统文化包装成事实保证；你的脚本用独立研究和自己的场景支撑。

## 未来调用规则

小五狼适合做认知课式的讲解主账号：当主题需要**让观众快速听懂一个概念、获得清晰框架并立刻想行动**时优先调用。写稿时只选一个结构模型；如需辅助能力，最多补充两项，并以你自己的主题、经历、场景和事实材料完成表层表达。具体语言约束以 `04_语言风格指纹.json` 为准：删除课堂口头禅、身份升级和为强调而重复的句子。
