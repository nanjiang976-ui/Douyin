# oMel｜统一文风说明书

## 账号来源与适用范围

- 账号名：oMel
- 账号 ID／主页链接：未提供
- 样本：19 条用户提供 SRT，18 条独立正文（其中 1 条完整重复）
- 适用主题：个人成长、自控力、专注、习惯、认知、迷茫、行动、财富观、赚钱意识、职业选择、人际沟通。
- 内容形态：以“读完一本书，替观众提炼可改变生活的若干原则”为主；书不是结论，而是观点的权威入口。

## 核心写法

先把观众正在做的一个小动作、正在承受的一种痛苦，或不愿承认的失败，具体地说出来。紧接着把“我是不是不行”改写为更可处理的机制问题，例如环境、阻力、注意力经济、思考模式、金钱关系。随后以“我会让你获得什么”“这本书会告诉你怎么做”给出强而明确的收益，并用书名、作者、阅读原版/提炼精华等信息建立解释资格。

最稳定的结构模型是：

1. `痛点动作 → 机制追凶 → 书籍原则 → 环境/练习动作`：适用于专注、拖延、自控、内耗。
2. `身份修复 → 能力缺口 → 思维模型 → 自我练习`：适用于粗心、决策、认知与沟通。
3. `未来代价 → 反主流命题 → 财富/选择框架 → 长期行动`：适用于赚钱、职业与人生选择。
4. `筛选观看者 → 让留下者获得身份 → 精华承诺 → 编号知识地图`：适用于长讲书、习惯和成长类主题。

## 开头：oMel 最值得学习的部分

### 稳定功能路径

`截停 → 照见 → 改写归因 → 欠答案/给收益`

- 截停：用“你明明知道”“我知道你现在”“如果你只有”“有个残酷事实”等句子，直接进入状态，不寒暄。
- 照见：给身体动作和时间细节，如大拇指下滑、通知带走注意力、工资刚到月中花完、躺床刷视频。
- 改写归因：用“不是你的错”“你并不笨，只是缺少……”把人格自责转成机制或能力缺口。
- 欠答案/给收益：明确说出下一步能得到的能力，而不是只说“会有收获”。

### 五个可调用开头家族

1. **身份修复型**：`你并不{人格否定}，也不是不想{目标}，只是缺少{可训练能力}。`
   - 适用：粗心、拖延、内耗、表达、起步难。
   - 关键：先给尊严，才有资格指出问题。

2. **动作矛盾型**：`你明明知道{更重要的事}，但为什么{身体动作/习惯}停不下来？`
   - 适用：手机、拖延、消费、熬夜、注意力。
   - 关键：用看得见的动作证据，不给空泛人格标签。

3. **此刻窥视型**：`我知道你现在正{平台上的具体状态}，也知道你看到{内容}想划走。`
   - 适用：横版知识、长视频、需要观众投入时间的内容。
   - 关键：承认观看成本，再给一个可以验证的收益或即时动作。

4. **反向筛选型**：`如果你还没准备好{改变/行动}，可以先划走；等你真想{目标}再回来。`
   - 适用：长内容、练习型内容、愿意投入的核心受众。
   - 关键：不是激将辱骂；要把留下者定义成愿意为自己负责的人。

5. **未来代价型**：`如果{错误方向/资源问题}不解决，{未来时间尺度}最痛的会是{具体代价}。`
   - 适用：财富、职业选择、长期习惯。
   - 关键：未来损失后必须迅速给出机制与现实路径，不能用恐惧逼留。

## 论证、节奏与情绪

- 论证先给强判断，再用书籍、人物、故事、数字或日常场景解释；最终一定回到一个普通人能做的动作。
- 语速感来自“短判断/短问句 → 较长解释 → 例子 → 一句回收”。问句总是承担下一拍，不是装饰。
- 正文以编号知识点推进；每一点最好只有一个独立机制，后面配一个场景或故事。
- 情绪路线通常是：`焦虑或羞耻 → 被准确看见 → 获得一个新解释 → 愿意行动`。财富主题会多一层“焦虑 → 反叛 → 进攻感”。
- 结尾常回到书籍推荐、重复关键观点和对观众的祝福，CTA较轻。

## 迁移边界

- 应迁移：具体场景开场、身份修复、问题机制化、反向筛选、即时操作承诺、短成本/长收益对比。
- 可以检索：原句、案例、人物、数字、比喻、叙事顺序；调用时保留素材 ID 与来源。
- 发布前必须核验：书籍作者/销量/引文、心理学与脑科学、投资或财富数据、医疗故事、人物经历。
- 质量控制：不要用“必然改变人生”“一定财富自由”“最痛苦”等无法兑现的保证替代真实收益；更好的落点是“帮你识别 X、完成 Y、开始 Z”。
