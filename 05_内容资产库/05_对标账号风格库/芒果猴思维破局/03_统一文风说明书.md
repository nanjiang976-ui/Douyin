## 一句话辨识

先用一句带风险、命运、身份或悖论色彩的强判断，把观众熟悉的善良、坦诚、懂事、自卑、等待或满足重新命名成一种隐性代价；随后连续放入具体场景、概念名和高记忆度比喻，最后给出可立即执行的边界、话术、身体动作或最小行动。

它真正擅长的不是“说狠话”，而是完成这条心理路径：

`先让我担心自己正在吃亏 → 再让我在生活细节里认出自己 → 给这个困境一个能记住的名字 → 让我获得自保或行动的许可 → 用一句更高价值的身份定义收住`

## 选题母题与观众痛点

高频母题集中在四组：

1. 社交防御：好心被反噬、说话泄底、被套话、不会拒绝、过度坦诚。
2. 身份与位置：穷相、配得感、自卑、开口索取、向上/同层/向下的角色表达。
3. 低谷与能力：贫穷时失去动力、沉迷短视频、觉得自己不适合口播或不会使用新工具。
4. 哲理与行动：满足为何导致麻木、等待为何比做错更耗能、主动如何增加反馈和确定性。

观众常被放在一个“明明出于好意或想变好，却因方法错误反而吃亏”的位置。文案先替他放大损失，再提供一套能收回控制权的解释。

## 开头机制

第一拍通常不铺背景，而直接完成一次价值翻转：

- 禁令型：禁止一个看似正常的动作，并预告关系、利益或机会代价。
- 命运因果型：把困境归因于一个被忽略的行为或身份姿态，再承诺翻盘收益。
- 悖论公式型：用对仗、数字差或三段口诀制造逻辑缺口。

第二拍必须让观众照见自己，常用面试、同事提问、亲友抱怨、借钱、奢侈品店、刷手机、等待联系客户等具体场景。第三拍再承诺解释一个机制或给一套话术/动作。没有场景和正文兑现的狠话，不属于可调用能力。

## 四个结构模型

### 1. 风险警告－机制追凶－边界防守

适合职场沟通、套话、关系边界、拒绝和信息保护。

完整路径是：危险法则 → 高频场景 → 对方真实动机或自己的情绪按钮 → 中计后果 → 防守台词/动作 → 体面或成熟收束。典型正文不是只讲“人性复杂”，而是把问题写成一个可演示的小剧场，再让防守动作与陷阱一一对应。

### 2. 身份刺痛－损失账单－系统重装

适合自卑、配得感、气场、开口索取、财富意识和社交位置。

完整路径是：身份判断 → 现实损失/反差人物 → 底层机制 → 多个生活自测 → 角色切换 → 编号动作或位置策略 → 尊严修复。它会先刺痛，但结尾必须把观众从被审视者改写为玩家、使用者或有权提出需求的人。

### 3. 低谷自证－障碍重命名－微小闭环

适合低谷、没有动力、技能学习、内容创作和 AI 辅助成长。

完整路径是：羞耻困境 → 创作者同起点 → 两类障碍命名 → 反驳宏大翻盘或天生能力 → 个人试验 → 筛选条件/最小闭环 → 立即行动。个人故事的作用是证明“我也从不会开始”，不是展示成功身份。

### 4. 悖论公式－系统解释－行动仪式

适合快乐、节制、主动、等待、不确定性和行动力。

完整路径是：悖论公式 → 极端比喻 → 心理或系统机制 → 普通生活镜像 → 重新计算得失 → 小仪式 → 格言或诗性收束。比喻承担解释功能，不能只负责显得深刻。

## 论证、案例与素材功能

- 生活场景：把抽象损失变成“我真的做过”的瞬间，是观众照见自己的主证据。
- 个人故事：提供同起点和后果，不承担普遍事实证明。
- 心理学、神经科学、生物学和古语：在来源中常承担“给判断加重量”的功能，但不应直接照搬；新稿需要核验，或换成更诚实的机制假设。
- 游戏、黑洞、货币、自动售货机等比喻：贯穿一段长解释，把关系或行动问题转成可见系统。
- 示例台词：在信息防御和拒绝主题中承担第一次正文兑现，必须围绕新场景重新写。
- 数字：用于把成本变得可计算；没有证据时不能保留原数字。

## 语言、节奏与情绪

账号与观众的距离像一个已经看穿规则的强势同辈，频繁直呼“你”，先下判断再解释。强度主要来自词义、反差和代价，而不是感叹号；15 份正文中没有中文感叹号，却有 47 个问号。

句子偏长，中位句长约 34 个中英文数字字符；一段平均约 2.89 句。稳定节奏是“长句把后果和画面铺满 → 短句命名机制 → 反问逼观众确认 → 金句封口”。问句多用于让观众回忆、替观众反驳或从机制转方法。

常见情绪路线：

- 警觉/后怕 → 识破 → 获得拒绝许可 → 从容。
- 窘迫/委屈 → 被准确命名 → 去魅 → 配得和占位。
- 羞耻/绝望 → 被理解 → 看见微动作 → 掌控和行动。
- 反常识好奇 → 机制被讲通 → 重新评价处境 → 珍惜或行动。

## 结尾功能

结尾很少只求点赞关注，更多承担四种任务：

1. 收权：闭嘴、停止替别人背责任、守住信息和资源。
2. 行动：发信息、搜索、开口、练一个句子、做一个低风险试探。
3. 身份修复：把观众从软弱、闯入者或失败者改写成有边界的玩家。
4. 价值升高：以成熟、慈悲、命运可能性或诗性画面缓和前文攻击性。
